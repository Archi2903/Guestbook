# Guestbook 
> Stack: Laravel 7.3, PHP 7.2.5|^8.0, MySQL 8.0
## Локальная установка
- composer install
- cp .env.example .env
- php artisan key:generate
- Прописать DB_DATABASE .env
- php artisan migrate:fresh 
- php artisan serve
- Посмотреть сайт можно по адресу http://r92043hv.beget.tech/
